Solarplus

Location: Humboldt County

Data Visualization and Analysis

Solarplus provides a means by which the live sensor data currently being monitored at the building in question, can be streamed into the system which would then perform analyses and visualizations to give facilities managers the relevant information necessary for the proper functioning of the building and its environment

Hardware Configuration Management

Solarplus also provides a means in which a user is able to interact with certain devices to change its setpoint values as well as change its schedule.
